# Phase 11 - UI / UX

## Objectif

Renforcer la presentation visuelle et l'experience utilisateur du portfolio avant les phases d'optimisation et de deploiement final.

## Direction visuelle

- Couleur principale : bleu-fonce.
- Couleur secondaire : bleu-clair.
- Couleur de base : blanc.
- Interface sobre, lisible et professionnelle.
- Animations courtes et utiles.

## Ameliorations realisees

- Accueil plus structure avec points forts.
- Hero d'accueil base sur les donnees admin : photo professionnelle, nom complet, titre professionnel et courte presentation.
- Cartes projets plus stables visuellement.
- Page admin avec avertissement d'etat.
- Focus clavier plus visible.
- Responsive renforce sur mobile et ordinateur.
- Espacements et grilles ajustes.

## Contraintes

- Garder des cards simples avec rayon de bordure limite.
- Eviter les effets decoratifs lourds.
- Ne pas casser les fallbacks Supabase.
- Garder une navigation utilisable sur mobile.
- Garder les textes lisibles sans chevauchement.

## Checklist

- Build Vite valide.
- Pages principales lisibles sur desktop.
- Pages principales lisibles sur mobile.
- Navigation horizontale mobile sans debordement de page.
- Cards projets lisibles avec ou sans image.
- Page admin clairement marquee comme non protegee.
